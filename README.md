# dev# 🕹️ DevArcade

A browser-based arcade platform featuring 5 resume-worthy games, fully playable on desktop and mobile.

**Live Demo:** https://bhuvanaju107.github.io/devarcade/

---

## Games

| Game | Tech Demonstrated |
|------|------------------|
| 🧠 Tic-Tac-Toe | Minimax algorithm, game state management |
| 🐍 Snake | Canvas API, game loops, collision detection |
| 🃏 Memory Match | DOM manipulation, animations, event handling |
| 🚀 Space Shooter | OOP, Canvas rendering, physics |
| ⚡ Dev Quiz | Dynamic UI, scoring logic, leaderboard |

---

## Features

- 📱 Fully mobile-responsive with touch controls
- 🎮 On-screen D-pad for Snake, virtual buttons for Space Shooter
- 🖱️ Works on desktop with keyboard and mouse
- ⚡ Zero dependencies — pure HTML, CSS, JavaScript
- 🎨 Dark arcade aesthetic with smooth animations

---

## How to Run Locally

No build step needed. Just open the file:

```bash
git clone https://github.com/YOUR-USERNAME/devarcade.git
cd devarcade
open index.html        # Mac
start index.html       # Windows
xdg-open index.html    # Linux
```
